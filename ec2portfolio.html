Thank you for visiting my portfolio website. I have created this website using knowledge I gained from studying to become a certified AWS Cloud Practitioner and fullstack developer. The portfolio website is hosted on a AWS EC2 instance.

In addition, this portfolio showcases several smaller projects I have created.

  • Ecommerce: Mock website for selling clothing apparel. The site allows for admin to create a profile, signin, or signout. Admin can add, edit, and delete products from the inventory. A customer can add products to their cart, view cart, and edit it. Uses HTML, CSS, JS, Node JS, and Express.js.
  • Movie Fight: A movie comparison project that allows the user to search for two different movies and then compares the scores (eg. oscars, IMBD rating, etc) to show which movie is the winner. Uses HTML, CSS, JS, Node JS, and an API.
  • Nutrition Label: Illustrates typography with the use of HTML and CSS.
  • Timer: A timer that default starts at 30 seconds, but allows the user to edit the about of time as desired. Uses HTML, CSS, and JS.
  • Quiz: The development of this quiz focused on web accessibility. The application is developed for most all users with the use of code that optimzes support of screen readers, smooth scrolling, and color contrast. Uses HTML and CSS.
  • Survey Form: The survey application explores the use of various types of html elements commonly used or specific to forms, such as form, fieldset, label, input (ie. name, email, number, radio, and checkbox), textarea, and button. Uses HTML and CSS.
